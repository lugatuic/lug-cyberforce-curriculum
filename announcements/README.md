## How to Use

1. **For a new announcement:** Copy relevant template
2. **Fill in:** Dates, links, deadlines, details
3. **Post to:** Discord + email (important announcements) or just Discord (engagement)
4. **Save:** Rename to `[year]-[event]-announcement.md` and commit to repo
5. **Archive:** Keep for reference + to copy from next year

---

## Key Rules

- Email = official, important deadlines (registration, competition day, C-Suite due)
- Discord = engagement, updates, Q&A, less formal
- Both = critical information (so nobody can say they didn't know)
- Always include: deadline, link (if applicable), contact for questions
- Use emoji for scannability
- Keep it under 200 words if Discord-only, longer is OK for email
